# <img src="https://raw.githubusercontent.com/Anto426/Anto426/main/asset/icon.gif" width="60px" /> About;

```sh
root@anto426: ~/vscodetheme (main⚡)$ cat theme_details.txt

A custom VS Code theme designed to mirror the rofi control menu palette:
  - Dark, premium glass-like surfaces
  - Compact contrast with soft borders
  - Dynamically updated wallpaper-driven highlight accent

Generated theme file target path:
  themes/Anto426-Rofi-Dynamic.json
```

> [!NOTE]
> The Arch installer builds this extension and the dotfiles color engine automatically rewrites the theme file whenever `~/.config/anto426/wallpaper_effects.sh` regenerates your wallpaper palette.

# <img src="https://raw.githubusercontent.com/Anto426/Anto426/main/asset/icon2.gif" width="70px" /> Build;

```sh
root@anto426: ~/vscodetheme (main⚡)$ yarn build
```

Install packages and compile:

```bash
yarn install --frozen-lockfile
yarn build
```

# <img src="https://raw.githubusercontent.com/Anto426/Anto426/main/asset/icon3.gif" width="70px" /> Local Install;

```sh
root@anto426: ~/vscodetheme (main⚡)$ yarn package --install
```

Package the `.vsix` extension file and install system-wide:

```bash
yarn package
code --install-extension anto426-vscode-theme-*.vsix --force
```
